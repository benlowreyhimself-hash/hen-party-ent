## Section Details

### 1. Hero Section
- **Height**: 400px
- **Background**: Property main image
- **Overlay**: Purple at 60% opacity
- **Text**: White, centered, large
- **Content**: Title + Location

### 2. Location Information Card
- **Background**: Light gray card
- **Border**: Subtle border
- **Padding**: Generous (p-6)
- **Info**: Location, Postcode, Address, Region

### 3. Photo Gallery
- **Layout**: 3 columns (desktop), 1 column (mobile)
- **Height**: 300px per image
- **Style**: Rounded corners, object-cover
- **Max**: 3 photos displayed

### 4. Description
- **Style**: Large text (text-lg)
- **Spacing**: Relaxed line height
- **Purpose**: Quick overview

### 5. Sales Content
- **Heading**: "Hen Party Life Drawing Entertainment"
- **Content**: 200-300 words
- **Style**: Prose styling, preserved whitespace
- **Purpose**: Compelling sales piece

### 6. Features List
- **Icon**: Purple checkmark (âœ“)
- **Style**: Bullet list with spacing
- **Purpose**: Highlight property amenities

### 7. Booking & Contact Section
- **Background**: Purple tint (primary/10)
- **Border**: Purple border
- **Content**: Contact info + booking button
- **GTM Tracking**: Phone and email clicks tracked

### 8. Google Maps
- **Link**: Opens in new tab
- **Style**: Primary color, underlined on hover

## Color Scheme

- **Primary**: Purple (#c87cc4)
- **Background**: White
- **Text**: Dark gray/black
- **Cards**: White with subtle borders
- **Accents**: Purple highlights

## Responsive Design

- **Mobile**: Single column, stacked sections
- **Tablet**: 2 columns for photos
- **Desktop**: 3 columns for photos, wider content

## SEO Elements

- Custom meta title
- Custom meta description
- Semantic HTML structure
- Proper heading hierarchy
- Alt text on all images
- Structured URLs
